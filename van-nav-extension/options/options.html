⚙️ Van Nav 扩展设置

配置您的导航站服务器连接

服务器配置

输入您的 Van Nav 服务器 IP 或域名
输入您的 Van Nav 服务器端口
在 Van Nav 后台的 API Token 管理中生成

扩展配置

开启后可使用快捷键快速打开扩展

⚡ 快捷键设置

如果快捷键无法使用，请点击下方链接手动设置：

  • Chrome 浏览器
  • Edge 浏览器
  • Brave 浏览器
  • Opera 浏览器
  • Vivaldi 浏览器

💡 建议：Ctrl+E (Win/Linux) 或 Command+E (Mac)

📖 使用说明

  1. 在 Van Nav 后台生成 API Token
  2. 填写服务器地址和端口
  3. 填写 API Token
  4. 点击"测试连接"确认配置正确
  5. 保存设置后即可使用扩展添加书签
